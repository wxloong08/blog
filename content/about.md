这个 blog 主要记录我在两类问题上的折腾：

- 怎么把 agent 工作流接得更顺
- 怎么把网页、论坛、问答页清洗成更适合模型阅读的输入

我长期关注的方向大致是：

- `Claude Code / Codex / MCP` 这一类 agent 工具链
- `SearXNG`、网页抓取、正文清洗、研究输入整理
- VPS、代理链路、基础设施和部署细节

这里不会刻意写成教程站，也不太想写成“资讯摘要”。  
更像是一个持续更新的工程笔记：把我实际遇到的问题、为什么默认方案不够、最后是怎么补上的，都尽量写清楚。

最近一段时间，我主要在做：

- 用 `SearXNG + searxng-mcp` 补齐 agent 的搜索能力
- 用 [url-reader](https://github.com/wxloong08/url-reader) 替代噪音过多的网页 reader
- 在 VPS 上维护这套工具链的运行环境
